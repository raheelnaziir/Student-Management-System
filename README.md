# StudentDB Manager — Project Overview

**StudentDB Manager** is a desktop application built in **Java** to manage student records efficiently.  
It demonstrates:

- A clean **Graphical User Interface (GUI)** using **Java Swing**  
- **Database Connectivity** with **MySQL** using **JDBC**  
- **Multi-Threading** with `SwingWorker` for smooth and responsive performance  

---

## Key Features

### ✓ CRUD Operations  
- Add new student records  
- View all students  
- Search student by ID  

### ✓ Responsive UI  
All database operations run **in the background** using `SwingWorker`, preventing the UI from freezing.

### ✓ Structured MVC Architecture  
The application follows **Model–View–Controller (MVC)** to maintain clean, scalable, and organized code.

---

## Prerequisites

### ✔ Java Development Kit (JDK)  
- Version **8 or higher**

### ✔ MySQL Server  
- Must be installed and running (default port **3306**)

### ✔ MySQL Connector/J  
- Required JDBC driver for Java  
- The `.jar` file must be added to your project's **classpath**
